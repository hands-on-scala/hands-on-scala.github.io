Basic pattern matching in Scala

published on May 06, 2016
Why should we care about pattern matching? Pattern matching is a quite commonly used “programming pattern” in functional languages. It’s somewhat similar to a sequence of if / else statements or to the switch statement of Java, but it is much more powerful, because it lets you decompose the object to be matched on the fly. This means that you can apply rules based on the matched object’s type or inner structure without having to manually disassemble the object into its parts. Read More...
